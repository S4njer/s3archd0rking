# s3archd0rking
This is a tool that search in a lot of search engines and export the results to .json, then you can view it with tools, like this: https://jsonviewer.stack.hu/

## Dorking Tool
The Dorking Tool is a versatile script that allows you to perform dorking, a technique used to find sensitive information or vulnerabilities on the internet, across multiple search engines. It automates the process of querying various search engines with a given query and gathers the search results for further analysis.

![d0rkinS3arch](https://www.riskinsight-wavestone.com/wp-content/uploads/2020/01/Capture.png)

## Features
Supports a wide range of search engines, including Google, DuckDuckGo, Bing, Dogpile, Google Scholar, Webopedia, Yahoo, Internet Archive, Baidu, Ask, AOL, Yandex, Excite, Lycos, Yippy, Startpage, and Gigablast.
Retrieves search results in a structured manner, providing a comprehensive overview of the information available on different search engines.
Saves the search results in a JSON file, allowing easy storage, analysis, and integration with other tools.
Supports the use of proxies for enhanced privacy and anonymity while performing dorking.
Usage
Run the script and provide the query you want to search for.
The script will automatically perform the search across multiple search engines, utilizing the provided query.
Once the dorking process is complete, the search results will be saved in a file named results.json.
Analyze the gathered data or use it in conjunction with other tools for further exploration.
Note: Ensure that you have the necessary dependencies installed and configured before running the script. You may need to modify the script if you want to add additional search engines or customize the result extraction logic.
